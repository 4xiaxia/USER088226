# 新版用户界面说明

## 概述
这个目录包含了新版用户界面的组件，包括：
1. 登录页面 (NewLogin.tsx)
2. 欢迎对话页面 (WelcomeDialog.tsx)
3. 主应用控制器 (NewApp.tsx)
4. 界面切换开关 (UISwitch.tsx)

## 如何启用新界面
新界面默认是禁用的。要启用新界面，有两种方法：

### 方法一：通过界面开关
1. 在浏览器中打开应用
2. 点击右上角的"旧界面/新界面"切换按钮
3. 切换到新界面后，页面会自动刷新

### 方法二：通过localStorage
1. 打开浏览器开发者工具
2. 在控制台中执行以下命令：
   ```javascript
   localStorage.setItem('enable_new_ui', 'true');
   ```
3. 刷新页面

## 组件说明

### NewApp.tsx
主应用控制器，负责管理用户界面的不同状态：
- 登录状态
- 欢迎对话状态
- 主应用状态

### NewLogin.tsx
新的登录页面，包含：
- 微信登录按钮
- 支付宝登录按钮
- 视觉效果和动画

### WelcomeDialog.tsx
登录后的欢迎对话页面，包含：
- 渐进式消息显示
- 功能介绍卡片
- 开始体验按钮

### UISwitch.tsx
界面切换开关，允许用户在新旧界面之间切换。

## 开发说明
所有组件都使用Tailwind CSS进行样式设计，并遵循项目的颜色和设计规范。
